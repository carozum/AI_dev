Azure Services

Azure Load Balancer

Supports high availability and scalability. A load balancer is designed to distribute incoming network traffic across multiple instances of applications or services, ensuring high availability and scalability. It can be configured to distribute traffic across availability zones within a region, providing redundancy in case of failures and allowing applications to handle increased loads while maintaining optimal performance and availability.

Azure Logic Apps

Logic apps are the way to connect various services, applications, and data sources, both within and outside of Azure, using workflow-oriented and integration scenarios.

Azure Resource Manager

ARM is a way to organize and manage azure resources using a unified group-based approach rather than deploying Azure items individually

Azure Virtual Network

Enable a user to control virtual machines and cloud services but are not specific to availability and scale.
