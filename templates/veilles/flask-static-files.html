Static files

They are css and js files.

  1. Create the 'static' folder and inside 2 other folders : css and js where you can put your css stylesheet and your js scrips.
  2. In the html templates, add link to the css stylesheet : href="/static/css/styles.css"
  3. Before the closing body tag, add the script tag with src='static/js/app.js'
  4. Create your own palette in styles.css
  5. Change the font family
  6. Uses icons if it makes sense
