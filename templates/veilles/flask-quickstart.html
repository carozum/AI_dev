Request Response Cycle

The request - response cycle is where a user requests information from your site, your site determines how to handle the request and what information to send back, and then a response is sent to the user.

Flask Quickstart


                from flask import Flask
            
                app = Flask(__name__)
            
                @app.route('/')  # request
                def index():
                    return "hello from Pet Adoption"  # response
            
                @app.route('/set-environment')
                def environment():
                    return "Set your environment"
            
                if __name__ == '__main__':
                    app.run(debug=True, port=8000, host='127.0.0.1')

Namespaces__name__ is __main__ or the name of the file ex __app__

First request/response, first route: a route is a command to run a specific function which then returns the response to the user.

Decorator : decorators are function that wrap around other functions and let you do things with them.

In the terminal, python3 app.py to run the flask app.
