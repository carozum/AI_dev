Templates

Templates are html files that flask uses to create the structure of our website.

  1. Add a folder called 'templates' - exact name
  2. Inside of this folder, we can create html file for our homepage : index.html.
  3. Add content to html file
  4. in app.py, import the render_template method and return render_template('index.html')

Flask code


            from flask import Flask, render_template
        
            app = Flask(__name__)
        
            @app.route('/')
            def index():
                return render_template('index.html')  
        
            @app.route('/pets')  # request
            def pets():
                return render_template('pets.html')
        
            @app.route('/set-environment')
            def environment():
                return render_template('set-environment.html')
        
            if __name__ == '__main__':
                app.run(debug=True, port=8000, host='127.0.0.1')
